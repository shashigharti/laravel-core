# Core Module for CMS
## Backend API(Laravel)
###### **Install Dependecies:**
`composer install` 

###### **Install Passport (API Authentication):**
`php artisan passport:install`

###### **Run Migrations:**
`php artisan migrate`
`php artisan db:seed `
`php artisan robust:migrate-all`


##FrontEnd (React)
https://github.com/shashigharti/react-core

######**Install packages:**
`npm install`

######**To start the app:**
`npm start`
